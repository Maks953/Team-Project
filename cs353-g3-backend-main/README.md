# CS353 G3 Backend

This is our back-end Django API server. In order to run this server you will need to pass a lot of enviromental variables:

- EMAIL_HOST=
- EMAIL_HOST_USER=
- EMAIL_HOST_PASSWORD=
- MONGO_HOST=
- MONGO_PASSWORD=
- MONGO_PORT=
- MONGO_USER=
- POSTGRES_HOST=
- POSTGRES_NAME=
- POSTGRES_PASSWORD=
- POSTGRES_PORT=
- POSTGRES_USER=
